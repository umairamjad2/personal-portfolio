# Portfolio Website

A modern **personal portfolio web application** built using **React** and **Vite**. This project showcases my projects, skills, and contact information with a clean, responsive, and interactive interface.

## 🚀 Features

* Showcases personal projects and experience
* Responsive design for desktop and mobile devices
* Modern UI and smooth navigation
* Component-based architecture using React
* Hot Module Replacement (HMR) support with Vite
* ESLint for code quality and consistency

## 🛠️ Technologies Used

* React.js
* Vite
* JavaScript 
* HTML5
* Tailwind CS
